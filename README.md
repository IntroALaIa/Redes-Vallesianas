# Redes Bayesianas

Este proyecto implementa un modelo de redes bayesianas utilizando Python y la librería pgmpy.

## Instalación

### Requisitos del Sistema
- **Python**: Versión 3.6 o superior (recomendado 3.8 o 3.9 para mejor compatibilidad).
- Si no tienes Python instalado, descárgalo desde [python.org](https://www.python.org/downloads/).

### Instalación de Dependencias
Instala las dependencias necesarias ejecutando el siguiente comando:

```bash
pip install pgmpy pandas networkx matplotlib
```

O, si prefieres usar un archivo de requerimientos, ejecuta:

```bash
pip install -r requerimientos.md
```

Nota: El archivo `requerimientos.md` contiene la lista de dependencias.

## Uso

1. Navega al directorio del proyecto.
2. Ejecuta el script principal o los notebooks según corresponda.

## Estructura del Proyecto

- `data/`: Datos utilizados en el proyecto.
- `notebooks/`: Notebooks de Jupyter para análisis y visualización.
- `src/`: Código fuente, incluyendo `model.py` que define el modelo de red bayesiana.
